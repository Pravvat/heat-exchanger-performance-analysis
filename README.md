# Heat Exchanger Performance Analysis

A Python-based chemical engineering mini project focused on analyzing the thermal performance of a heat exchanger under different operating conditions using engineering calculations and data visualization.

---

## Objective

The objective of this project is to study the effect of temperature and flow rate on heat exchanger efficiency and heat transfer performance.

---

## Technologies Used

- Python
- MS Excel
- Matplotlib
- Heat Transfer Fundamentals

---

## Concepts Used

- Heat Transfer Rate
- LMTD (Log Mean Temperature Difference)
- Heat Exchanger Efficiency
- Energy Balance
- Flow Rate Analysis

---

## Features

- Calculates heat transfer performance
- Analyzes efficiency variation with flow rate
- Generates graphical visualization of results
- Demonstrates basic thermal engineering calculations

---

## Project Workflow

1. Define inlet and outlet temperatures
2. Calculate heat transfer rate
3. Compute LMTD values
4. Analyze efficiency trends
5. Visualize results using graphs
